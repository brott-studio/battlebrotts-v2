# 🤖⚔️ BattleBrotts v2

> Build. Program. Fight.

Build autonomous combat Brotts, teach them how to fight with drag-and-drop Behavior Cards, and watch them battle in top-down arenas.

**Engine:** Godot 4 | **Platform:** Web (HTML5) | **Genre:** Auto-battler / Strategy
